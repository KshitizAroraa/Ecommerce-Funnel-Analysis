# E-Commerce Funnel Revenue Analysis

An end-to-end analytics project analyzing a 6-month synthetic electronics e-commerce dataset.
Covers funnel drop-off, revenue metrics, cohort retention, and LTV using Python, SQL, and Tableau.

---

## Project Structure

```
ecommerce-funnel-analysis/
├── data/
│   ├── raw/
│   │   └── ecommerce.db          # SQLite database (generated by notebook 01)
│   └── processed/
│       ├── users.csv
│       ├── products.csv
│       ├── sessions.csv
│       ├── events.csv
│       ├── orders.csv
│       ├── order_items.csv
│       ├── orders_enriched.csv   # orders + session + user joined (for Tableau)
│       └── *.png                 # all generated charts
├── notebooks/
│   ├── 01_data_generation.ipynb  # generate synthetic data + SQLite DB
│   ├── 02_eda_and_cleaning.ipynb # data quality checks + EDA charts
│   ├── 03_funnel_analysis.ipynb  # funnel drop-off, by device/channel/age
│   ├── 04_revenue_analysis.ipynb # revenue metrics, AOV, top products
│   └── 05_cohort_analysis.ipynb  # retention heatmap, LTV curves
├── sql/
│   ├── schema.sql                # table definitions + indexes
│   ├── funnel_analysis.sql       # funnel queries
│   ├── revenue_metrics.sql       # revenue, AOV, discount queries
│   └── cohort_analysis.sql       # cohort retention + revenue queries
├── tableau/
│   └── dashboard_notes.md        # field mappings + sheet design guide
├── requirements.txt
└── README.md
```

---

## Dataset

Synthetic 6-month electronics e-commerce data (2024-01-01 to 2024-06-30)

| Table | Rows | Description |
|-------|------|-------------|
| users | 5,000 | User profiles with country, device, channel, age |
| products | 120 | Electronics across 9 subcategories |
| sessions | ~40,000 | User sessions with device and channel |
| events | ~100,000+ | Funnel events per session |
| orders | ~5,000+ | Completed, returned, cancelled orders |
| order_items | ~6,000+ | Line items per order |

**Product subcategories:** Smartphones, Laptops, Tablets, Headphones, Smart Watches, Cameras, TVs, Gaming, Accessories

---

## Funnel Stages

```
Page View → Product View → Add to Cart → Checkout Start → Purchase
```

---

## Key Metrics Analyzed

- Funnel conversion and drop-off rates at each stage
- Breakdown by device type, acquisition channel, and age group
- Weekly funnel trends over 6 months
- Top cart abandonment products
- Monthly revenue, AOV, and order volume
- Revenue by channel, device, and product subcategory
- Discount impact on AOV and revenue
- Repeat purchase rate
- Monthly cohort retention heatmap
- Cumulative LTV curves per cohort

## How to Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run notebooks in order
Open each notebook in VS Code and run all cells (Shift+Enter or Run All):

```
01_data_generation.ipynb   → creates SQLite DB + CSVs
02_eda_and_cleaning.ipynb  → data quality + EDA charts
03_funnel_analysis.ipynb   → funnel visualizations
04_revenue_analysis.ipynb  → revenue charts
05_cohort_analysis.ipynb   → cohort + LTV analysis
```

### 3. Build Tableau dashboard
- Open Tableau Public (free) or Tableau Desktop
- Connect to CSVs in `data/processed/`
- Follow the guide in `tableau/dashboard_notes.md`
